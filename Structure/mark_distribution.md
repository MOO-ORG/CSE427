# Marks Distribution (Tentative)

## Overall Breakdown

| Component            | Weight | Notes                                                 |
| --------------------- | :----: | ------------------------------------------------------ |
| Attendance            |  10%   | See [Attendance Marking](#attendance-marking) below     |
| Class Test / Quiz     |  15%   | n−1 counted, usually best 3 out of 4                   |
| Midterm               |  25%   |                                                          |
| Final                 |  30%   |                                                          |
| Lab                   |  20%   | See [Lab Marks Breakdown](#lab-marks-breakdown) below   |

## Attendance Marking

Applies uniformly across all sections:

- **90% or above** → 10 marks
- **Below 70%** → 0 marks
- **Between 70% and 90%** → `4 + ((Attendance % − 70) / 20) × 6`

  Examples: exactly 70% → 4, exactly 75% → 5.5, exactly 80% → 7, exactly 85% → 8.5, and so on.

## Lab Marks Breakdown

Lab classes are **weekly, not bi-weekly**. Total lab marks: 20.

| Component        | Marks | Details                                                                                                                                                                                                                                                    |
| ------------------ | :---: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Lab Attendance    |   3   | Introduced this semester for the first time. Try to attend all lab classes — missing more than 2 lab classes throughout the semester results in 0 out of 3.                                                                                              |
| Lab Assignment    |   5   | 6–8 lab assignments this semester, **all of which count**. Submissions are taken centrally via Google Forms.                                                                                                                                              |
| Lab Project       |  12   | The majority of lab marks. Runs from before the midterm week until before the final week. Work in groups of 2–3: finalize a topic and dataset, study related literature, implement the proposed idea, and submit a recorded presentation (in English) plus a project report. The report accounts for 50% of the project marks and must use the IEEE double-column LaTeX template. |

All the best.
